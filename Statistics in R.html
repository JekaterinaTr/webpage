R programming language:

Statistical analysis and visualization

The R programming language is a powerfull tool for statistical analysis with a vast choice of statistical models and powerfull visualizations.

Here are a few examples of how I used R for statistical analysis and graphical representations. In the first graph (Figure A) I tried to learn if a certain market tomato price is statistically different from the previously measured price using one-sample t-test. In another example, I wanted to find out if there is a statistically significant difference between salaries of different genders using independent samples t-test (Figure B). Finally, an outcome of a biological data statistical analysis can be represented with a heatmap (Figure C).
