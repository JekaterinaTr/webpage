Excel data analysis project:

Travel insurance sales

In this data analysis project I analized statistics of travel insurance sales from several travel agencies and airlines using exlusively Excel.

The original data was carefully cleaned up: the duplicates were removed, every data was checked for inaccuracies or typos. For exalmple, trips with travel duration lower then one day doesn't make any sence so these were removed as obvious mistakes

The statistics were calculated using Excel formulas and pivot table options. The results were represented in an interactive dashboard here.

Data sources: Travel insurance dataset from Kaggle (https://www.kaggle.com/datasets/ersany/travel-insurance/data). Icons used: ‘Sales’ icon Made x Made, ‘Comission’ icon by Soremba, ‘Insurance claims’ icon by Siti Masriatun, ‘Travel’ icon by Iconmas from Noun Project
